CoDiFy is a modern quiz application built for web developers to strengthen their fundamentals through concept-driven practice. Users can choose a technology, attempt quizzes, and instantly see their performance in a clean and responsive interface.

---

## Features

- Technology-based quizzes (JavaScript, React, etc.)
- Concept-focused questions
- Instant score calculation
- Performance feedback (correct vs wrong)
- Retry quiz without page reload
- Responsive design (mobile + desktop)
- Clean dark UI

---

## Tech Stack

- React (Vite)
- React Router DOM
- Context API
- Tailwind CSS
- lucide-react (icons)
- Vercel (deployment)

---


---

## How It Works

1. User opens the app and selects a technology.
2. The quiz loads dynamically based on the selected technology.
3. User answers the questions (one answer per question).
4. On submit, the system compares answers with the correct ones.
5. Score and performance are shown instantly.
6. User can retry the quiz or choose another technology.
